# NEAT Snake #
NEAT (NeuralEvolution of Augmenting Topologies) is a machine learning technique that alters the weights an biases, along with the structure of the network itself.

This legacy project is an implementation of the technique on a game of snake made in PyGame.

## Functions ##
### app.py ###
This code allows you to train an agent. Adjust paramaters in the config-feedforward file to experiment.

### data_visualisation ###
Run this code after training a model to see a graph.

## test_winner ##
Run this code after training a model to see the best model play a game of snake
